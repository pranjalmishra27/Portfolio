## 🚀 Overview

This repository contains the source code for my frontend personal playground and resume. The website is designed to be highly responsive, developer-centric, and features lightning-fast load times with custom interactive animations.

## 🛠️ Built With

The entire site is built to be as lightweight and performant as possible without relying on bulky frontend frameworks:
* **HTML5**
* **CSS3** (Custom properties, Flexbox/Grid layouts, Keyframe Animations)
* **Vanilla JavaScript** (Intersection Observers for scroll reveals, physics-based cursor tracking, dynamic DOM rendering)
* **Zero external dependencies/frameworks**
